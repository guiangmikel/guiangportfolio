# Deploy Your First Website to the Cloud

This project contains a simple three-page website intended for deployment to a free cloud hosting platform.

## Pages
- `index.html` — Home page
- `about.html` — About Me page
- `contact.html` — Contact page

## Features
- Student full name placeholder
- Course and year level placeholder
- Self-introduction content
- Simple responsive design

## Deployment Options

### GitHub Pages
1. Create a GitHub repository.
2. Add these files and commit them.
3. On GitHub, go to `Settings` > `Pages`.
4. Select the `main` branch and `/ (root)` folder.
5. Save and open the published site URL.

### Netlify
1. Create a free Netlify account.
2. Click "Add new site" and choose "Deploy manually".
3. Drag and drop this project folder into Netlify.
4. Wait for the deployment to finish.
5. Open the generated Netlify URL.

### Render
1. Create a free Render account.
2. Choose "Static Site" and connect your GitHub repository.
3. Set the build command to none and publish directory to `/`.
4. Deploy and visit the Render URL.
